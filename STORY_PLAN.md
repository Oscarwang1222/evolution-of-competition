# STORY-PLAN

故事大纲：《竞争的演化》

📄 飞书文档：https://tcntzj2zhtf5.feishu.cn/wiki/IoCPwHwJOiuEkEkHNfCcDPM5nxg

一个类似《信任的进化》的互动叙事游戏，玩家跟随一群普通人的演化，亲历内卷的诞生、巅峰与崩溃，最终理解创造力为何是破局之道。

整体结构

每章格式统一：
叙事漫画/对话 — 推进剧情，展示人物处境
玩家关键选择 — 决定角色走向（影响该章节模拟的初始状态）
模拟演示 — 展示群体行为，用数字说话
教育小结 — 点明本节主旨

第一章：乌托邦

剧情设定

所有人都处于最初的"普通人"状态。没有竞争，没有淘汰，每个人的精力 n=10，每轮把 n-2=8 点用于分数，2 点浪费（睡觉、娱乐）。

普通人：每轮 +8 分数
分数曲线：线性增长，平稳、安全、无聊

叙事

旁白："这是一个小镇。每个人都一样——早上起床，晚上睡觉，偶尔担心考试，但没什么大不了。"
"日子，就这样一天天过去……"
"直到有一天，来了一个……卷王。"

模拟演示

展示：10个普通人，20轮
图表：分数线性增长，缓慢而稳定
关键词：**稳定 = 缺乏动力**

教育落脚点

稳定不是坏事，但**没有淘汰的公平，也会让进步停滞**。这不是坏，这是人性。

第二章：第一个卷王

剧情设定

小镇来了一个转学生——**卷王**。他每轮把所有精力 n=10 全用于分数。普通人（8分/轮）永远打不过他。

旁白："他的出现，打破了一切宁静。"

玩家第一次选择

"你是小镇的原住民。你发现自己的分数被卷王碾压了。"
"你会怎么做？"

选项 A：跟着卷 → 进化为"卷王"，进入第三章
选项 B：保持普通人 → 模拟演示后被淘汰，重新选择

模拟演示（选B触发）

场景：1个卷王 vs 9个普通人，20轮
图表：卷王一骑绝尘，普通人缓慢跟随
关键数据：第20轮，卷王分数 ≈ 200，普通人 ≈ 160
结果：普通人因为分数太低，被系统判定为"不适合继续"，淘汰出局

教育落脚点

"你不卷，时代也会推着你卷。" 当有人开始卷，其他人只有两个选择：跟，或者被淘汰。

第三章：群体觉醒（多米诺骨牌倒下）

剧情设定

旁白："就这样，一个卷王变成了一群卷王。所有人都在跑，但你发现——大家都在跑，你就还是在原地。"
"当所有人都变成了卷王，新的问题出现了……"

玩家选择

"你看到大家都在卷。原定的游戏规则是：分数排名最后的被淘汰。"
"但现在所有人分数都一样高（都≈200+），排名几乎相同……"
"系统决定改变规则。"

选项 A：继续卷，想办法比别人的分数更高 → 进入第四章
选项 B：寻找另一种可能…… → 直接进入第五章

模拟演示

场景：全部都是卷王，30轮
图表：**所有人的分数曲线完全重叠**——大家一样努力，排名依然存在（谁先卷谁在前），但相对差距不变
关键转折：精力惩罚开始出现——连续5轮剩余精力小于2，精力上限 n 开始衰减

教育落脚点

当所有人都在用同样的策略竞争，**个体努力变成了群体内耗**。每个人都更累了，但没有人真正赢。

第四章：代价来临

剧情设定

所有人都变成了卷王。但系统开始显现它的残酷：精力不是无限的。连续透支，最终会累垮。

旁白："所有人都已经是卷王了。但你开始听到一些……抱怨。"
"有人说：'我分数在涨，但我也越来越累了。'"
"有人说：'这样下去不是办法……'"

玩家关键选择

"这时候，系统做了一个重大宣布："
"**为了筛选出真正的强者，我们引入一种新的评分机制——总分**"

选项 A：继续纯卷 → 角色不变
选项 B：寻找另一种可能……

模拟演示

场景：全部卷王，50轮（精力惩罚上线）
图表：
  - **前20轮**：分数指数级上升（每个人都+10/轮）
  - **第20~25轮左右**：精力惩罚开始（有人 n→9）
  - **第30轮后**：n 持续衰减，总分增长放缓，最终停滞
关键数据：当所有人都精力耗尽（n=1），每轮只+1分，社会彻底停滞

教育落脚点

**内卷是有天花板的**——精力是有限的，但分数的追求是无限的。当精力耗尽，所有人的努力都在原地踏步。**唯分数论的弊端开始显现**。

第五章：破局

剧情设定

在所有人都筋疲力尽的时候，有人开始问：**有没有另一种方法？**
一个叫"创哥"的人出现了。他不跟大家比分数，他把精力分出一部分去"搞创新"。
"卷王赢了我的分数，但赢不了我的**创造力**。"

玩家选择

"创哥的路线看起来很奇怪——他几乎不学习，只搞创新。"
"周围的卷王都在嘲笑他：'你分数这么低，以后肯定没出息！'"
"但创哥说：'你们不懂，分数只是一时的，我的创造力会让我超越你们！'"

选项 A：跟着创哥走 → 进入创哥路线
选项 B：继续当卷王 → 进入卷王路线

模拟演示（只看应试分数）

场景：10个卷王 vs 1个创哥，只按应试分数排名
前30轮：创哥因为几乎不做应试，分数垫底，被所有人嘲笑
第30轮后：创哥开始感受到压力——"明明我这么努力，为什么分数还是最低？"
结果：创哥因为分数太低，被系统淘汰出局

旁白："这是一个关于'唯分数论'的教训。创哥的创造力固然强，但在只看分数的系统里，他连展示自己的机会都没有。"

教育落脚点

**唯分数论是有缺陷的**——它只看眼前，不看潜力；只看应试，不看创造。创哥被淘汰了，但他的创造力并没有消失——只是被系统埋没了。

第六章：新公式

剧情设定

旁白："创哥的淘汰在小镇引起了轰动。有人开始质疑：'分数真的能代表一切吗？'"
"终于有一天，系统宣布了一个新规则："
"**我们决定引入一种新的评分方式——总分**"
"这个总分，不只看分数，还看**创造力**。"

新公式揭示

旁白："其实，系统一开始隐瞒了一个规则——"
"**总分 = 应试分 + 创造力 × 系数**"
"其中，系数 = min(应试分 / 100, 3)"
"这意味着什么？"
- 应试分越高，系数越大，创造力的乘数效应越明显
- 但如果应试分太低（≈50），系数只有0.5左右，创造力的加成就被严重削弱了

玩家选择

"你现在明白了：创哥的创造力虽高，但因为应试太低，系数被压低，总分上不去。"
"那么，你会怎么选？"

选项 A：选择创哥路线（纯创造） → 角色变为"纯创哥" → 因基础不牢被淘汰
选项 B：寻找更优方案 → 进入第七章（均衡发展）

模拟演示（新公式）

场景：10个卷王 vs 10个纯创哥，50轮（用新公式计算总分）
前20轮：创哥自信满满："我的创造力会让我翻身！"
第20~35轮：创哥发现不对——"为什么我总分还是这么低？！"
第35轮后：创哥绝望——"系数只有0.5，我的创造力被严重低估了！"
最终结果：纯创哥总分 ≈ 200，卷王总分 ≈ 275，创哥再次被淘汰

旁白："看到了吗？纯创哥再次被淘汰了。不是因为他不优秀，而是因为**应试基础太弱**，拖了创造力的后腿。"

教育落脚点

**纯创造也有问题**——没有扎实的基础，创造力无法充分发挥。就像没有地基的房子，盖不高。

第七章：均衡发展

剧情设定

旁白："纯创哥再次被淘汰后，小镇陷入沉默。"
"难道就没有一条路，能兼顾分数和创造力吗？"
"这时候，一个叫'均衡哥'的人站了出来。"
"他说：'我有一个办法——**应试和创造，我都要**。'"

玩家选择

"均衡哥的方案是：每轮把精力分成两部分，一部分用来保证分数不至于太低，一部分用来积累创造力。"
"这意味着他不会是最强的卷王，也不会是最纯粹的创哥。"
"但他相信，**两条腿走路，总比一条腿稳**。"

选项 A：跟着均衡哥走 → 进化为"均衡创哥"
选项 B：继续当卷王 → 角色不变

模拟演示

场景：10个卷王 vs 10个均衡创哥，60轮
前20轮：卷王领先（应试碾压），均衡创哥稳步积累
第20~40轮：均衡创哥创造力开始显现，总分逐渐追上
第40轮后：指数效应叠加，**均衡创哥全面超越卷王**
第60轮：均衡创哥总分 ≈ 800，卷王总分 ≈ 275

教育落脚点

**全面发展才是正解**——应试够线 × 创造力积累 = 指数爆发。只有两条腿走路，才能走得更远。

第八章：新秩序

剧情设定

世界分化成了四类人：
- 继续纯卷的卷王（精力耗尽，停滞）
- 试图纯创的创哥（应试太低，上限被锁死，两次被淘汰）
- 均衡发展的创哥（前期艰难，最终登顶）
- 全面发展的顶级创哥（双轨并行，指数碾压）

模拟演示

场景：混合场景——纯卷王 + 纯创哥（被淘汰）+ 均衡创哥 + 顶级创哥，70轮
图表：
  - 纯卷王：前快后慢，最终停滞在 275左右
  - 纯创哥：**两次被淘汰**（第一次被唯分数论淘汰，第二次被公式淘汰）
  - 均衡创哥：逐渐加速，第40轮超越纯卷王
  - 顶级创哥：指数爆炸，第50轮后一骑绝尘

玩家选择

"故事到这里，你要选择你的结局："

- 继续当纯卷王 → 结局：疲惫，中途被均衡创哥超越
- 走纯创哥路线 → 结局：两次被淘汰
- 走均衡发展路线 → 结局：前苦后甜，最终超越纯卷王
- 全面发展顶级创哥 → 结局：一骑绝尘

最终章：教育总结

四种人生轨迹的对比

| 策略 / 代表人物 / 短期(10轮) / 中期(30轮) / 长期(60轮) / 结局 |
| --- |
| 纯卷王 / 卷王 / 第1 / 第2 / 衰退 / 被超越 |
| 纯创哥 / 创哥 / 垫底 / 淘汰 / — / 两次被淘汰 |
| 均衡发展 / 创哥（均衡型） / 第4 / 第2 / 登顶 / 超越纯卷王 |
| 全面发展 / 顶级创哥 / 第3 / 第1 / 指数碾压 / 孤独的领跑者 |

核心主旨

- 唯分数论有缺陷：只看应试，创哥连展示的机会都没有
- 纯创造也有问题：应试基础太弱，创造力被系数压低，上限锁死
- 全面发展才是正解：应试够线 × 创造力积累 = 指数爆发
- 选择比努力更重要：不是比谁更卷，而是比谁更会平衡

尾声

"故事到这里，你已经看到了四种人生。"
"纯卷王，累但停滞；纯创哥，两次被淘汰。"
"只有全面发展的人，才能真正登顶。"
"但这不是结局——这是开始。"
"你会选择哪条路？"

附录：技术设计

模拟层（simulation.ts）

保持独立纯函数，不依赖 UI：
- runSimulation(roles, rounds, initialEnergy) → RoundResult[]
- getFinalRanking(results) → Player[]
- getScoreHistory(results) → ScoreHistory[]

叙事层（narrative.ts）

负责章节配置和叙事结构：
- 每章的剧情文字
- 章节对应的模拟参数（角色列表、轮数、精力）
- 玩家选择 → 角色类型映射
- 关键数据的解读文案

UI层（App.tsx）

- 漫画展示组件（图片+对话）
- 选择按钮
- 模拟可视化（图表+排名）
- 教育总结页

数据结构

```
interface Chapter {
  id: number
  title: string
  narrative: NarrativeBlock[]  // 图片+对话
  choice: Choice[]              // 选项列表
  simulationConfig: SimConfig  // 模拟参数
  education: string            // 本章小结
}

interface NarrativeBlock {
  type: 'text' | 'image' | 'dialogue'
  content: string
  character?: RoleType
}
```
