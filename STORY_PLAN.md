# 故事大纲：《竞争的演化》

> 📄 飞书文档：https://tcntzj2zhtf5.feishu.cn/wiki/IoCPwHwJOiuEkEkHNfCcDPM5nxg

> 一个类似《信任的进化》的互动叙事游戏，玩家跟随一群普通人的演化，亲历内卷的诞生、巅峰与崩溃，最终理解创造力为何是破局之道。

---

## 整体结构

每章格式统一：

1. **叙事漫画/对话** — 推进剧情，展示人物处境
2. **玩家关键选择** — 决定角色走向（影响该章节模拟的初始状态）
3. **模拟演示** — 展示群体行为，用数字说话
4. **教育小结** — 点明本节主旨

---

## 第一章：乌托邦

### 剧情设定
所有人都处于最初的"普通人"状态。没有竞争，没有淘汰，每个人的精力 n=10，每轮把 n-2=8 点用于分数，2 点浪费（睡觉、娱乐）。

```
普通人：每轮 +8 分数
分数曲线：线性增长，平稳、安全、无聊
```

### 叙事
> 旁白："这是一个小镇。每个人都一样——早上起床，晚上睡觉，偶尔担心考试，但没什么大不了。"
> "日子，就这样一天天过去……"
> "直到有一天，来了一个……卷王。"

### 模拟演示
- 展示：10个普通人，20轮
- 图表：分数线性增长，缓慢而稳定
- 关键词：**稳定 = 缺乏动力**

### 教育落脚点
稳定不是坏事，但**没有淘汰的公平，也会让进步停滞**。这不是坏，这是人性。

## 第二章：第一个卷王

### 剧情设定
小镇来了一个转学生——**卷王**。他每轮把所有精力 n=10 全用于分数。普通人（8分/轮）永远打不过他。

> 旁白："他的出现，打破了一切宁静。"

### 玩家第一次选择
> "你是小镇的原住民。你发现自己的分数被卷王碾压了。"
> "你会怎么做？"
- **选项 A：跟着卷** → 进化为"卷王"，进入第三章（群体觉醒）
- **选项 B：保持普通人** → 模拟演示后被淘汰，重新选择角色

### 模拟演示（选B触发）
- 场景：1个卷王 vs 9个普通人，20轮
- 图表：卷王一骑绝尘，普通人缓慢跟随
- **关键数据**：第20轮，卷王分数 ≈ 200，普通人 ≈ 160
- **结果**：普通人因为分数太低，被系统判定为"不适合继续"，淘汰出局

### 教育落脚点
"你不卷，时代也会推着你卷。" 当有人开始卷，其他人只有两个选择：跟，或者被淘汰。

## 第三章：群体觉醒（多米诺骨牌倒下）

### 剧情设定
> 旁白："就这样，一个卷王变成了一群卷王。所有人都在跑，但你发现——大家都在跑，你就还是在原地。"
>
> "当所有人都变成了卷王，新的问题出现了……"

### 玩家选择
> "你看到大家都在卷。原定的游戏规则是：分数排名最后的被淘汰。"
> "但现在所有人分数都一样高（都≈200+），排名几乎相同……"
> "系统不得不改变规则。"
>
> **你会怎么做？**
- **选项 A：继续卷，想办法比别人的分数更高** → 进化为"超级卷王"
- **选项 B：寻找另一种可能** → 进入第四章（代价来临）

### 模拟演示
- 场景：全部都是卷王，30轮
- 图表：**所有人的分数曲线完全重叠**——大家一样努力，排名依然存在（谁先卷谁在前），但相对差距不变
- **关键转折**：精力惩罚开始出现——连续5轮剩余精力<2，精力上限 n 开始衰减

### 教育落脚点
当所有人都在用同样的策略竞争，**个体努力变成了群体内耗**。每个人都更累了，但没有人真正赢。

---

## 第四章：卷到极致（代价来临）

### 剧情设定
所有人都变成了卷王。但系统开始显现它的残酷：精力不是无限的。连续透支，最终会累垮。

> 旁白："所有人都已经是卷王了。但你开始听到一些……抱怨。"
> "有人说：'我分数在涨，但我也越来越累了。'"
> "有人说：'这样下去不是办法……'"

### 玩家关键选择
> "这时候，系统做了一个重大宣布："
> "**为了筛选出真正的强者，我们引入一种新的评分机制——总分**"
>
> **你会怎么做？**
- **选项 A：继续纯卷，告诉自己再撑一下就好了**
- **选项 B：寻找另一种可能……**

### 模拟演示
- 场景：全部卷王，50轮（精力惩罚上线）
- 图表：
  - **前20轮**：分数指数级上升（每个人都+10/轮）
  - **第20~25轮左右**：精力惩罚开始（有人 n→9）
  - **第30轮后**：n 持续衰减，总分增长放缓，最终停滞
- **关键数据**：当所有人都精力耗尽（n=1），每轮只+1分，社会彻底停滞

```
第50轮：平均精力 ≈ 2~3，平均分数 ≈ 180~200（几乎不再增长）
```

### 教育落脚点
**内卷是有天花板的**——精力是有限的，但分数的追求是无限的。当精力耗尽，所有人的努力都在原地踏步。**唯分数论的弊端开始显现**。

---

## 第五章：破局

### 剧情设定
在所有人都筋疲力尽的时候，有人开始问：**有没有另一种方法？**

一个叫"创哥"的人出现了。他不跟大家比分数，他把精力分出一部分去"搞创新"。

> "卷王赢了我的分数，但赢不了我的**创造力**。"

### ⚠️ 机制转折：新总分公式登场

**系统宣布新的总分计算方式**：
```
总分 = 应试上限系数 × 创造力加成 + 创造力分
应试上限系数 = min(应试分 / 100, 3)  // 应试分数越高，总分上限越高
```

**这意味着**：
- 应试分 = 100时，系数=1.0，总分上限 = 1.0 × 创造力 + 创造力 = 2倍创造力
- 应试分 = 200时，系数=2.0，总分上限 = 2.0 × 创造力 + 创造力 = 3倍创造力
- 应试分 = 50时，系数=0.5，总分上限 = 0.5 × 创造力 + 创造力 = 1.5倍创造力（很低！）

**创哥的困境**：每轮只用1分应试，7分创造，50轮后应试≈50，创造力≈350。但因为应试太低，系数只有0.5，总分上限被卡死了！

### 玩家选择
> "你注意到一件事：创哥的路线很奇怪——"
> "他的创造力很高，但应试分太低了，导致他的总分上限被锁死。"
> "而卷王虽然创造力=0，但应试分高，系数高，总分反而更高……"
>
> **你会怎么做？**
- **选项 A：继续纯卷** → 角色不变
- **选项 B：寻找"应试够用 + 创造力积累"的均衡路线** → 进化为"创哥（均衡型）"

### 模拟演示
- 场景：10个卷王 vs 1个均衡创哥，50轮
- **前20轮**：卷王碾压（系数高，创造力0起步）
- **第20~35轮**：创哥创造力积累，但应试太低导致上限被卡
- **第35轮后**：**关键发现**——创哥的应试太低（≈100），系数只有1.0，而均衡创哥应试≈180，系数1.8，总分开始反超！
- **最终结果**：均衡创哥应试≈200（系数=2.0），创造力≈200，总分 ≈ 2.0×200 + 200 = **600**；纯卷王应试≈275（系数=2.75），但创造力=0，总分只有275

```
纯卷王（50轮）：应试≈275，创造力≈0，系数=2.75，总分≈275
纯创哥（50轮）：应试≈50，创造力≈350，系数=0.5，总分上限≈525（但实际发挥受限）
均衡创哥（50轮）：应试≈200，创造力≈200，系数=2.0，总分≈600
```

### 教育落脚点
**唯分数论（只看应试）是有缺陷的**——但纯创造也有问题，因为应试太低会锁死总分上限。**只有找到均衡点，才能突破天花板**。

---

## 第六章：创新者的困境

### 剧情设定
均衡创哥路线被越来越多人看到。有人开始模仿——但不是每个人都能坚持。

> "均衡发展的方法需要耐心。前期你要保证应试不拖后腿，同时积累创造力——这比纯粹卷王要累得多。"
> "有人中途放弃了：要么回去纯卷（被淘汰），要么变成纯创哥（也被淘汰）。"
> "但真正坚持下来的人……成为了**全面发展的顶级创哥**。"

### 玩家选择
> "均衡路线需要双重投入：应试要够线，创造要积累。周围的人要么纯卷要么纯创，你却在两条线上同时跑。"
> "你要中途放弃，还是继续？"
- **选项 A：放弃（降级为纯卷王 or 纯创哥）** → 淘汰出局
- **选项 B：坚持到底** → 进化为"全面发展的顶级创哥"

### 模拟演示
- 场景：10个卷王 vs 10个全面发展的顶级创哥，60轮
- **前20轮**：卷王领先（应试碾压），顶级创哥应试≈180（系数1.8）
- **第20~40轮**：顶级创哥追平（创造力系数开始显现）
- **第40轮后**：指数效应叠加，顶级创哥全面碾压
- **第60轮**：顶级创哥总分≈800+，是卷王≈500的**1.6倍**

### 教育落脚点
**创新最大的敌人不是竞争者，而是过程中的孤独和怀疑**。全面发展的顶级创哥揭示了一个真相：创造力一旦越过临界点，加上扎实的基础，总分是卷王永远无法追赶的。

## 第七章：新秩序

### 剧情设定
世界分化成了四类人：
- 继续纯卷的卷王（精力耗尽，停滞）
- 试图纯创的创哥（应试太低，上限被锁死）
- 均衡发展的创哥（前期艰难，最终登顶）
- 全面发展的顶级创哥（双轨并行，指数碾压）

### 模拟演示
- 场景：混合场景——纯卷王 + 纯创哥 + 均衡创哥 + 顶级创哥，70轮
- 图表：
  - 纯卷王：前快后慢，最终停滞在 500左右
  - 纯创哥：**上限被锁死**（应试≈50，系数=0.5，总分≈350）
  - 均衡创哥：逐渐加速，第35轮超越纯卷王
  - 顶级创哥：指数爆炸，第50轮后一骑绝尘

### 玩家选择
> "故事到这里，你要选择你的结局："
- **继续当纯卷王** → 结局：疲惫，中途被均衡创哥超越
- **转型纯创哥** → 结局：上限被锁死，无法突破
- **走均衡发展路线** → 结局：前苦后甜，最终超越纯卷王
- **全面发展顶级创哥** → 结局：一骑绝尘

## 最终章：教育总结

### 四种人生轨迹的对比

| 策略 | 代表人物 | 短期(10轮) | 中期(30轮) | 长期(60轮) | 结局 |
|------|----------|-----------|-----------|-----------|------|
| 纯卷王 | 卷王 | 🥇 第1 | 🥈 第2 | 衰退 | 被超越 |
| 纯创哥 | 创哥 | 垫底 | 瓶颈 | 上限被锁死 | 无法突破 |
| 均衡发展 | 创哥（均衡型） | 第4 | 第2 | 🥇 登顶 | 超越纯卷王 |
| 全面发展 | 顶级创哥 | 第3 | 🥇 第1 | 🥇 指数碾压 | 孤独的领跑者 |

### 核心主旨

1. **纯内卷有天花板**：精力有限，卷到极致后所有人都在原地踏步
2. **唯分数论有缺陷**：只看应试，创哥的上限被锁死；但纯创造也不行，上限同样受限
3. **全面发展才是正解**：应试够线 × 创造力积累 = 指数爆发
4. **选择比努力更重要**：不是比谁更卷，而是比谁更会平衡

### 尾声
> "故事到这里，你已经看到了四种人生。"
> "纯卷王，累但停滞；纯创哥，上限被锁。"
> "只有全面发展的人，才能真正登顶。"
> "但这不是结局——这是开始。"
> "你会选择哪条路？"

## 附录：技术设计

### 模拟层（simulation.ts）
保持独立纯函数，不依赖 UI：
- `runSimulation(roles, rounds, initialEnergy) → RoundResult[]`
- `getFinalRanking(results) → Player[]`
- `getScoreHistory(results) → ScoreHistory[]`

### 叙事层（narrative.ts）
负责章节配置和叙事结构：
- 每章的剧情文字
- 章节对应的模拟参数（角色列表、轮数、精力）
- 玩家选择 → 角色类型映射
- 关键数据的解读文案

### UI层（App.tsx）
- 漫画展示组件（图片+对话）
- 选择按钮
- 模拟可视化（图表+排名）
- 教育总结页

### 数据结构
```ts
interface Chapter {
  id: number
  title: string
  narrative: NarrativeBlock[]  // 图片+对话
  choice: Choice[]              // 选项列表
  simulationConfig: SimConfig  // 模拟参数
  education: string            // 本章小结
}

interface NarrativeBlock {
  type: 'text' | 'image' | 'dialogue'
  content: string
  character?: RoleType
}
```
